# Handboek app toegankelijkheid

Stichting Appt werkt aan een wereld waarin iedereen apps gelijkwaardig kan gebruiken. Om deze visie werkelijkheid te maken, hebben wij een handboek geschreven. Het helpt je om op een eenvoudige manier de basisprincipes van app toegankelijkheid te leren.

Het Nederlandse handboek is ruim 1400 keer op papier verspreid, en duizenden keren digitaal. Na dit succes schrijven we een nieuwe versie met bijgewerkte informatie.

Met trots introduceren we binnenkort de nieuwe versie van ons handboek voor app-toegankelijkheid.

Wil je het handboek digitaal ontvangen? Laat je gegevens achter in ons [Handboekformulier](https://appt.li/handbook-form).
