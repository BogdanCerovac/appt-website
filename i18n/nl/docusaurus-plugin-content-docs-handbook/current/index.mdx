# Appt handboek toegankelijkheid

<TwoColumnBlock>
  <LeftColumn>
    <ColumnRow>
      <div>
        Stichting Appt werkt aan een wereld waarin iedereen apps gelijkwaardig kan gebruiken. Om deze visie werkelijkheid te maken, hebben we een handboek geschreven. Het helpt je om op een eenvoudige manier de basis van app toegankelijkheid te leren.

        Op 23 juni 2021 hebben we de eerste druk van het handboek gepubliceerd. Deze versie is 1.500 keer op papier verspreid en duizenden keren digitaal.

        Op 16 mei 2024 hebben we de tweede druk van het handboek gepubliceerd. De inhoud is verbeterd en we hebben uitleg toegevoegd over de nieuwe succescriteria uit WCAG 2.2.
