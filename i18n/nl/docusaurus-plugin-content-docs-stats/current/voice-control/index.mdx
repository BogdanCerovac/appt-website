# Stembediening

Met stembediening is het mogelijk om een app te bedienen met je stem. Er verschijnt een raster op het scherm met allerlei cijfers of labels. Door deze cijfers of labels uit te spreken kun je bepaalde acties uit voeren.

Op Android toestellen is **Voice Access** te installeren om apps via je stem te bedienen. Bij iOS apparaten is stembediening standaard ingebouwd, dit heet **Voice Control**.
