# Linkdoel (in context)

Zorg dat het doel van elke link duidelijk is. Gebruikers kunnen een lijst van links maken. De link en zijn omliggende tekst moet context geven waar je naar toe navigeert. Duidelijke links zijn handig voor iedereen en met name voor gebruikers van hulpmiddelen.

## Impact

Gebruikers kunnen met hulpmiddelen een overzicht van links op het scherm opvragen. Het moet direct duidelijk zijn waar elke link naartoe verwijst.

## Controleren

“*Is het voor elke link duidelijk wat het doel is?*“

Dit kan getest worden zonder hulpmiddelen.
