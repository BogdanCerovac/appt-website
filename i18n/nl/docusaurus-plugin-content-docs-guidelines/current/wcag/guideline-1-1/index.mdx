# Tekstalternatieven

Alle inhoud moet ook als tekst zijn beschreven. Tekst kan namelijk zowel visueel, auditief of braille worden weergegeven. Gebruikers kunnen hierdoor informatie omzetten in een vorm die aansluit bij hun behoeften.
