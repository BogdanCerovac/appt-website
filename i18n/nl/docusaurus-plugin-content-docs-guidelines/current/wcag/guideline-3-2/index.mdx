# Voorspelbaar

Het moet voorspelbaar zijn wat er na een actie gebeurt. Voorkom dat het scherm verspringt bij focus of invoer, dit kan verwarrend zijn. Houd de navigatie op dezelfde plek op elk scherm. Zorg dat de icoontjes op elk scherm dezelfde functie hebben. Hierdoor weten gebruikers waar ze kunnen navigeren en wat ze kunnen verwachten.
