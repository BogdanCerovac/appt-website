Zorg dat reeds verstrekte gegevens niet opnieuw worden gevraagd. Gebruikers met cognitieve beperkingen kunnen moeite hebben om te onthouden wat ze eerder hebben ingevoerd. Reeds ingevoerde informatie moet automatisch ingevuld worden, of beschikbaar zijn om te selecteren.
