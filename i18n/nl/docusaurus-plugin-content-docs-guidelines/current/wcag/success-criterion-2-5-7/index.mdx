# Sleepbewegingen

Zorg dat sleepbewegingen een gelijkwaardig alternatief hebben via een enkele aanwijzer zoals tikken of lang indrukken. Gebruikers kunnen onderdelen mogelijk niet slepen vanwege mobiliteitsbeperkingen of gebruik van hulpmiddelen. Een knop kan een alternatief zijn voor drag-and-drop.
