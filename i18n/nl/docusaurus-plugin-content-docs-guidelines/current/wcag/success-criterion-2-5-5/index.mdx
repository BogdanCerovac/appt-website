# Grootte van het aanwijsgebied (uitgebreid)

Zorg dat elementen groot zijn zodat ze gemakkelijk geactiveerd kunnen worden. Gebruikers met ernstige lichamelijke beperkingen kunnen middelgrote elementen niet activeren. De minimale grootte is is 44x44 pixels.
