# Grootte van het aanwijsgebied (minimum)

Zorg dat elementen groot genoeg zijn en voldoende afstand hebben zodat gebruikers ze makkelijk kunnen activeren. Gebruikers met lichamelijke beperkingen kunnen kleine elementen mogelijk niet activeren. Het minimale formaat is 24x24 pixels.
