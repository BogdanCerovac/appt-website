# Leesbaar

Inhoud moet leesbaar zijn met hulpmiddelen. Dit is vooral belangrijk voor hulpmiddelen die tekst naar spraak omzetten. De inhoud kan duidelijk worden voorgelezen indien de taal juist is ingesteld.
