# Parsen

Zorg ervoor dat de broncode van de app geen fouten bevat en geen verouderde functies gebruikt. Hulpmiddelen gedragen zich mogelijk niet zoals verwacht wanneer code niet voldoet aan moderne standaarden.

## Impact

Gebruikers worden buitengesloten worden wanneer een app niet (goed) werkt op hun toestel.

## Controleren

“*Zit de app technisch goed in elkaar?*“

We raden aan om regelmatig een (externe) code review uit te voeren.
