# Op tijd gebaseerde media

Iedereen moet toegang hebben tot media, zoals audio en video. Mensen die doof zijn kunnen niet horen wat er wordt gezegd. Mensen die blind zijn kunnen niet zien wat er op beelden getoond wordt. Door een transcript, ondertiteling en/of audiodescriptie aan te bieden kan iedereen media waarnemen.
