Inhoud op het scherm moet aanpasbaar zijn aan de behoeften van gebruikers. Door de structuur en informatie door te geven aan hulpmiddelen, kan deze op verschillende manieren worden gepresenteerd. Kopteksten worden bijvoorbeeld visueel aangegeven, maar moeten ook worden gemarkeerd voor schermlezer gebruikers.
