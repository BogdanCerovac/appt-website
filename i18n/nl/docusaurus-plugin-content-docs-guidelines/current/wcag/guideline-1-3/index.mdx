# Aanpasbaar

Inhoud op het scherm moet zich kunnen aanpassen aan de wensen van gebruikers. Dit bereik je door alle informatie op een softwarematige manier beschikbaar te maken. De informatie kan vervolgens op de gewenste manier worden gepresenteerd. Een schermlezer kan bijvoorbeeld voorlezen dat dikgedrukte tekst een koptekst is.
