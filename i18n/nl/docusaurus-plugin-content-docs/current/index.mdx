Elk platform biedt verschillende functies om een app toegankelijker mee te maken. We hebben codevoorbeelden geschreven voor meerdere platformen. Daarnaast hebben we handleidingen geschreven over de beschikbare toegankelijkheidsfuncties. Als je niet weet waar je moet beginnen, start dan met onze [beginnergids voor toegankelijke apps](/nl/artikelen/toegankelijke-app-beginnersgids).
